📋 사용 방법 (클릭하여 펼치기)

Thread에서 목록 복사하는 방법:

  1. 웹 브라우저에서:
    • threads.com에서 로그인
    • 프로필 → 팔로워/팔로잉 클릭
    • 직접 모두 드래그 선택 후 복사(ctrl(cmd)+c)

💡 복사한 텍스트는 "유저명 → 설명" 형식으로 되어있어도 자동으로 유저명만 추출됩니다.

⚠️ Thread 숫자 불일치 현상 설명

프로필 숫자와 실제 목록이 다른 이유:

1. 동기화 지연 (가장 흔함)

  • 프로필 숫자 = 캐시된 요약 데이터
  • 목록 = 실시간 데이터
  • → 업데이트 시점 차이로 불일치 발생

2. 계정 변동

  • 비활성화/삭제된 계정은 목록에서 즉시 사라짐
  • 하지만 전체 숫자에는 늦게 반영됨

3. 스팸/봇 정리

  • Meta가 주기적으로 가짜 계정 대량 삭제
  • 목록과 숫자 반영 시점이 다름

해결 방법:

  • ✅ 시간이 지나면 자동 해결 (대부분)
  • ✅ 앱 재시작 또는 새로고침
  • ✅ 캐시 삭제 (Android)
  • ✅ 앱 업데이트 확인
